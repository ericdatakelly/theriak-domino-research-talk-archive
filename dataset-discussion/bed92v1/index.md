## Erik Duesterhoeft
*2017-04-27 13:10 UTC*

I am making my database 'BED92.v1' available (including amphibole solid solution and melt models). This database is build on JUN92d.bs database (based on Berman, 1988 + modifications). Watch the header of the database to find a documentation of all changes. Comments are welcome !

You can download the file from ResearchGate <https://dx.doi.org/10.13140/RG.2.2.10285.69609>
or download my "New compiled version for MAC and WIN" (see <https://research-talk.org/viewtopic.php?f=5&t=28> )

For now, you can cite it as:
E. Duesterhoeft (2017), BED92.v1 - Theriak-Domino database, ResearchGate, doi: 10.13140/RG.2.2.10285.69609
