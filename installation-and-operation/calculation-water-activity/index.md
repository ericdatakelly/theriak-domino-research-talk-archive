In water systems reactions dependent from water activity. In case standart calculation TD aH2O - constant (and eq 1.0?).

How calculate isopleth for aH2O?
For complicated system correct algorithm (in my case):
Standart pseudosection-->approximate field-->isopleth aH2O for field-->find aH2O--> add aH2O in database (STEAM in JUN92b) --> Recalculate pseudosection.

In trend calculation algorithm be  same for each point.
